## 运算符



### 算术运算符

| 运算符 | 说明   | 示例  | 结果 |
| ------ | ------ | ----- | ---- |
| +      | 加     | 1+1   | 2    |
| -      | 减     | 1-1   | 0    |
| *      | 乘     | 2*3   | 6    |
| /      | 除     | 10/2  | 5    |
| //     | 整除   | 10//3 | 3    |
| %      | 取余   | 10%3  | 1    |
| **     | 幂运算 | 2**4  | 16   |

#### 优先级（从高到低）

* 第一级：**
* 第二级：*，/，%，//
* 第三级：+，-



### 赋值运算符 

=、+=、-=、*=等

### 比较运算符

\>、<、>=、<=、==，!=

### 逻辑运算符

and、or、not

### 位运算

&（位与）、|（位或）、^（位异或）、~（位取反）、<<（左移位）、>>（右移位）



### 运算符的优先级

| 运算符               | 说明             |
| -------------------- | ---------------- |
| **                   | 幂               |
| ~、+、 -             | 取反，正号和负号 |
| *、/、%、//          | 算术运算符       |
| +、-                 | 算术运算符       |
| <<、>>               | 左移和右移       |
| &                    | 位与             |
| ^                    | 位运算符中的异或 |
| \|                   | 位运算符中的位或 |
| <、<=、>、>=、!=、== | 比较运算符       |
